# Projektrichtlinien

## Sprache und Schreibweise

- **Umlaute korrekt schreiben**: Immer die korrekten deutschen Umlaute verwenden (ä, ö, ü, Ä, Ö, Ü) statt der Ersatzschreibweise (ae, oe, ue, Ae, Oe, Ue).
  - Richtig: `über`, `für`, `Größe`, `Menü`
  - Falsch: `ueber`, `fuer`, `Groesse`, `Menue`
- Das Eszett (ß) wird ebenfalls korrekt verwendet.
